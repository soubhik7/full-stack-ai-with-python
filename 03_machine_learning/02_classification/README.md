# 03.02 — Classification

> Predict which category an input belongs to.

## Notebooks (run in order)

| # | Notebook | Algorithm | Dataset |
|---|----------|-----------|---------|
| 1 | `classification_logistic_regression_churn.ipynb` | Logistic Regression | Telecom churn |
| 2 | `classification_knn_customer_categories.ipynb` | K-Nearest Neighbours | Customer categories |
| 3 | `classification_decision_trees_drug.ipynb` | Decision Tree | Drug recommendation |
| 4 | `classification_svm_cancer.ipynb` | Support Vector Machine | Cancer diagnosis |
| 5 | `01_mnist_beginner_explanation.ipynb` | Multi-class (conceptual) | MNIST digits |
| 6 | `02_mnist_performance_evaluation.ipynb` | Evaluation metrics deep-dive | MNIST digits |

## When to use which?

| Algorithm | Best when… |
|-----------|-----------|
| Logistic Regression | Linearly separable, need probabilities |
| KNN | Simple, no training time, small datasets |
| Decision Tree | Interpretability matters, mixed feature types |
| SVM | High-dimensional, clear margin |
