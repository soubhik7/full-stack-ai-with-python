# 03.03 — Clustering

> Find natural groupings in data **without labels**.

## Notebooks (run in order)

| # | Notebook | Algorithm | Best for |
|---|----------|-----------|---------|
| 1 | `clustering_kmeans_customer_segmentation.ipynb` | K-Means | Spherical clusters, known k |
| 2 | `clustering_hierarchical_cars.ipynb` | Hierarchical (Ward linkage) | Dendrogram, unknown k |
| 3 | `clustering_dbscan_weather.ipynb` | DBSCAN | Arbitrary shapes, noise/outlier handling |

## Key difference from classification

Classification needs labelled data. Clustering discovers structure with **no labels** — the algorithm decides what "similar" means based on distance.
