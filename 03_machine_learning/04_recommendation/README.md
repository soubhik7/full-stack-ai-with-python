# 03.04 — Recommendation Systems

> Suggest items a user is likely to enjoy.

## Notebooks

| # | Notebook | Approach | Requires |
|---|----------|----------|---------|
| 1 | `recsys_content_based_movies.ipynb` | Content-Based Filtering | Item features (genre, director, cast) |
| 2 | `recsys_collaborative_filtering_movies.ipynb` | Collaborative Filtering | User-item interaction history |

## The two strategies

**Content-based:** "This movie is similar to movies you liked" — compare item features.
**Collaborative:** "Users like you also liked this" — compare user behaviour patterns.

Most production systems use a hybrid of both.
