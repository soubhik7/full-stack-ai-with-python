# 17 — AI-901: Microsoft Azure AI Fundamentals

Exam-prep materials for **AI-901**, Microsoft's current fundamentals-level AI certification (the successor to AI-900, which retired June 30, 2026). AI-901 covers two domains: identifying AI concepts/capabilities (40–45%) and implementing AI solutions with **Microsoft Foundry** (55–60%) — generative AI, agents, speech, vision, and Content Understanding.

| File | Content |
|------|---------|
| [`EXAM_NOTES.md`](EXAM_NOTES.md) / `.pdf` | Full study notes — responsible AI, generative AI fundamentals, model deployment/configuration, and the Foundry portal/SDK/Agent Service/Content Understanding surface |
| [`EXAM_PRACTICE_QUESTIONS.md`](EXAM_PRACTICE_QUESTIONS.md) / `.pdf` | 60 original scenario-based practice questions with answers and explanations, weighted to match the official blueprint |

## A note on how this was built

These are **original practice materials**, not real exam questions. Real AI-901 items are protected by the Microsoft Certification Agreement's non-disclosure terms, and sites that crowd-source or sell "real" exam dumps are both violating that agreement and, typically, infringing Microsoft's copyright in the exam content — using them risks certification revocation and doesn't actually test understanding. Instead, `EXAM_NOTES.md` was built from the **public, official** [AI-901 skills-measured outline](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-901) and cross-checked against current Microsoft Learn documentation for Foundry, Foundry Tools, and Content Understanding; `EXAM_PRACTICE_QUESTIONS.md` is a fresh set of scenario questions written from those notes, distributed across domains in proportion to the exam's own weighting. This mirrors the same approach already used for [`14-azure-ai103/EXAM_PRACTICE_QUESTIONS.md`](../14-azure-ai103/EXAM_PRACTICE_QUESTIONS.md).

Product surfaces referenced here (Microsoft Foundry, Foundry Tools, Content Understanding) are moving fast in 2026 — treat exact portal navigation and SDK details as a snapshot from the exam's April 15, 2026 update, and verify anything you build against current Microsoft Learn docs. For officially sanctioned practice, use Microsoft's free Practice Assessment via AI Skills Navigator, linked from the [AI-901 exam page](https://learn.microsoft.com/en-us/credentials/certifications/exams/ai-901/).
